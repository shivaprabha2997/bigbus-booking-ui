# Bus Booking UI

Simple frontend UI inspired by RedBus homepage.

## Technologies

HTML
CSS
JavaScript
Maven (for packaging)

## Project Structure

src/main/webapp
    index.html
    css/style.css
    js/script.js

pom.xml

## How to Run

Option 1 (Simple)

Open index.html in browser.

Option 2 (Using Maven + Tomcat)

1. Install Maven
2. Run:

mvn clean package

3. Deploy generated WAR file in Tomcat.

target/bus-booking.war

## Features

Search bus UI
Responsive layout
Simple validation
